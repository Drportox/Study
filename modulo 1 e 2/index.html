Aula 9- Margins and Padding no CSS

Para n causar confusao, o padding e o espaçamento interno, e o margin e o espaçamento externo.

    
        .textodeexemplo {
            border: 1px solid green;
            padding: 10px;
            margin: 10px;
                 }

Se quiser editar como voce quer sua pagina sempre faca isso

    
        body {
            margin: 0px;
        }

Definir o espaçamento interno ou externo

Os exemplos serve para margin (externo) tambem

    
        padding-left: Xpx;
        padding-right: Xpx;
        padding-top: Xpx;
        padding-bottun: xpx;

        padding: Xpx Xpx Xpx Xpx;
        (resumindo os valores sao em ordem horaria:em cima,lado, baixo e outro lado)
        desse modo pode ganhar tempo, e seu codigo fica mais organizado e pratico.

Aula 10- Width e Height no CSS

Exemplo 1:criando um cabeçalho com a div

Esse 100% significa que pega o maximo tam possivel, funciona o mesmo com o height

    
        .cabeça {
            background-color: black;
            width:100%;
            height:100px;
        }

Exemplo 2:usando em uma imagem

Para iamgens o ideal e sempre deixar o width auto
outro ponto importante, a porcentagem e sempre em relacao ao tam da tela, se for 50% sera metade da tela

Exemplo 3:Herança

Aqui em resumo, vc quer colocar uma img dentro da div e deixar ela sincronizada com o tam da div, usa a herenca

    
    CODIGO HTML
        <div class="cabeça">
        <img src="https://www.google.com.br/logos/google.jpg" alt="logo google"/>
        </div>
    


    
    CODIGO CSS  
        body {
            margin:0;
                }
        .cabeça {
            background-color: black;
            width:500px;
            height:500px;
                 }
        img {
            width:inherit;
        }

Exemplo 4: Max width ou Min Width
