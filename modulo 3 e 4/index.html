            EDITANDO TABLE:
        Como pode ver, coloquei a tabela tam 600px e fiz uma borda solida de 1px preta.
                   
                    table {
                        width:600px;
                        border:1px solid black;
                    }
            EDITANDO COLUNAS TD TH:
        Quando fizemos o codigo acima, fica apenas em volta da tabela e nao de todo o conteudo
        entao, criamos uma folha de estilo para as colunas,so para lembrar td sao as linhas do
        tbody e th sao as colunas do thead:
        Nesse caso usei a mesma border, coloquei um padding para ter um espaçamento interno, mudei a cor para branco e coloquei o texto no centro.

                    td, th {
                        border:1px solid black;
                        padding:10px;
                        color: white;
                        text-align:center;
                    }
            EDITANDO CABEÇALHO DA TABELA:
        Bom esse codigo, usei para editar especificamente o cabeçalho da tabela, apenas mudando 
        a cor de fundo.

                    table thead {
                        background-color:rgb(12, 10, 10);
                    }

            EDITANDO CORPO DA TABELA:
        Aqui e p/ editar especificamente o corpo da tabela, apenas mudando a cor do fundo.

                    table tbody{
                        background-color:rgb(43, 12, 12);
                    }

            EDITANDO TBODY QUANDO PASSO O MOUSE EM CIMA:
        Nao tem como ser mais especifico, esse codigo e quando passamos o mouse em cima do item, e cria seu efeito que determinei aqui
        ou seja quando passar em cima do corpo da tabela, a cor mudara.
                    
                    tbody tr:hover { 
                        background-color:rgb(6, 20, 81);
                    }

            EDITANDO COR APENAS EM NUMEROS PARES:
        Isso nao foi explicado muito bem do porque esse motivo, mas o codigo é esse, mesmo e so vamos aceitar.
        as cores onde estiver os itens pares ficaram nessa que escolhi.
                    
                    tbody tr:nth-child(even) {
                        background-color: rgb(124, 95, 196);
                    }

            EDITANDO A FONTE DO TEXTO:
        Aqui as tabelas ja vem de modo geral, em negrito, entao criei esse atributo apenas para deixar normal.
                    
                    th {
                        font-weight:normal;
                    }
